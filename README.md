## Sections

- **Contact**: Contact information and social links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Grid/Flexbox)
- Vanilla JavaScript
- GitHub API
- Font Awesome icons

## Local Development

Simply open `index.html` in your browser to view the site locally.

## GitHub API Integration

The site automatically fetches and displays your public GitHub repositories using the GitHub REST API. No authentication is required for public repositories.

## Customization

To customize this portfolio for yourself:

1. Update the `GITHUB_USERNAME` in `script.js`
2. Modify the content in `index.html` (name, bio, experience, skills, etc.)
3. Update social media links
4. Adjust colors in `styles.css` (CSS variables in `:root`)
